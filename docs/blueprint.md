# **App Name**: Guía España

## Core Features:

- Panel de Control Personalizado: Un dashboard principal que ofrece acceso rápido a todas las secciones clave de la aplicación, mostrando el progreso de regularización del usuario a través de un sistema de 'porcentaje completado' almacenado localmente.
- Ruta de Trámites Interactiva: Guías paso a paso para trámites esenciales como Empadronamiento, asignación de NIE/TIE, Número de la Seguridad Social y Tarjeta Sanitaria (SIP/TSI), optimizadas para disponibilidad offline.
- Gestor de Formularios y Citas Oficiales: Acceso directo y botones para descargar formularios oficiales (EX-15, 790-012, EX-10) y enlazar a sedes electrónicas para solicitar citas previas (Extranjería, Policía), incluyendo una sección con 'Trucos para la Cita Previa'. Incluye un recordatorio para el Certificado Digital/Cl@ve Permanente y nota sobre la necesidad de Adobe Acrobat Reader para PDFs.
- Bóveda de Documentos (Checklist Arraigo): Lista de requisitos interactiva para trámites de 'Arraigo Social' y 'Arraigo Laboral' (actualizada a la última reforma), con simulación de carga de archivos y seguimiento del estado en el almacenamiento local del dispositivo.
- Directorio de ONGs y Recursos: Un directorio filtrable de organizaciones no gubernamentales (ej. Cruz Roja, CEAR, Cáritas) y oficinas públicas de ayuda, segmentado por ciudades de España (ej. Madrid, Barcelona, Valencia, Jaén).
- Soporte Multilingüe Avanzado: Arquitectura preparada para ofrecer toda la interfaz y el contenido de la aplicación en Español, Inglés, Francés, con etiquetas de botones traducidas también al Árabe para la descarga de formularios.
- Botón de Asistencia Urgente: Un botón de emergencia visible en la interfaz para acceso directo al 112 y una lista de contactos para asistencia jurídica gratuita, manteniendo toda la información confidencial a nivel local.

## Style Guidelines:

- Esquema de color principal: Azul institucional (#2E73BF) que inspira calma y confianza, reflejando el tono profesional de la aplicación.
- Color de fondo: Un blanco muy suave con un sutil matiz azul (#F3F6F9), garantizando alto contraste y una interfaz limpia y minimalista.
- Color de acento: Un cian vibrante y amigable (#29B2CB), utilizado para elementos interactivos y puntos clave de información, ofreciendo una diferencia clara del azul principal.
- Fuente principal y para titulares: 'Inter' (sans-serif), elegida por su legibilidad excepcional en pantallas móviles, su diseño moderno y su neutralidad que evitan el desorden visual, combinando con el estilo minimalista.
- Iconografía: Se utilizarán los 'Lucide Icons', eligiendo aquellos que simbolicen claramente las acciones y contenidos, como el icono de 'Archivo/PDF' para la descarga de formularios. El estilo será claro, nítido y moderno para facilitar la comprensión.
- Disposición 'Mobile-first': El diseño de la interfaz estará totalmente optimizado para dispositivos móviles, priorizando la facilidad de uso, con grandes áreas táctiles, fuentes claras y alto contraste. Los elementos se presentarán de forma limpia y directa para evitar el desorden visual y mejorar la accesibilidad.
- Animaciones: Se incorporarán transiciones sutiles en la navegación y micro-interacciones al completar un paso o al alternar estados, proporcionando una experiencia de usuario fluida y receptiva sin distracciones.
